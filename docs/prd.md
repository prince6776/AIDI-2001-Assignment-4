# Student To-Do App PRD

## Overview
This project is a small student to-do web app built for an assignment demonstration. The app helps a user manage simple tasks in a clean and minimal interface.

## Goals
- Build a very small working app.
- Demonstrate a structured AI-assisted workflow.
- Keep implementation simple enough to finish quickly.

## In Scope
- Add a new task
- Mark a task as completed
- Delete a task
- Filter tasks by All or Completed

## Out of Scope
- User accounts
- Cloud database
- Due dates
- Categories or tags
- Collaboration
- Notifications

## User Stories
- As a user, I want to add a task so I can track something I need to do.
- As a user, I want to mark a task completed so I can track progress.
- As a user, I want to delete a task so I can remove items I no longer need.
- As a user, I want to filter tasks so I can view all tasks or only completed ones.

## Acceptance Criteria

### Add Task
- User can type a task into an input box.
- User can add the task to the list.
- Empty tasks are not added.

### Complete Task
- User can click a task or checkbox to mark it completed.
- Completed tasks are visually different from active tasks.

### Delete Task
- User can remove a task from the list.
- Deleted tasks disappear immediately.

### Filter Tasks
- User can switch between All and Completed views.
- The list updates correctly based on the selected filter.
